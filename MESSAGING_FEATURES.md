# نظام الرسائل المتكامل - Synaptic Space

## نظرة عامة

تم بناء نظام رسائل متقدم يضاهي Discord و WhatsApp و Telegram مع ميزات فريدة إضافية.

---

## الميزات المتاحة

### 1. التفاعلات (Reactions)
- **6 تفاعلات مخصصة** بصور فريدة
- تفاعلات فورية مع realtime updates
- عرض أسماء المتفاعلين عند hover
- إمكانية إلغاء/تغيير التفاعل
- تجميع التفاعلات المتشابهة مع عداد

**كيفية الاستخدام:**
- مرر الماوس على الرسالة
- اضغط على أيقونة 😊
- اختر التفاعل المناسب

---

### 2. الردود (Replies)
- الرد على أي رسالة بنقرة واحدة
- معاينة الرسالة الأصلية
- ربط مباشر بين الرسالة والرد
- إمكانية إلغاء الرد قبل الإرسال

**كيفية الاستخدام:**
- مرر الماوس على الرسالة
- اضغط على أيقونة الرد
- اكتب ردك واضغط Shift+Enter

---

### 3. الإشارات (Mentions)
- إشارة للأعضاء باستخدام @اسم_المستخدم
- قائمة منسدلة تلقائية عند الكتابة
- بحث ذكي في الأسماء وأسماء المستخدمين
- إشعارات فورية للمستخدمين المُشار إليهم
- تمييز بصري للرسائل التي تحتوي إشارة لك

**كيفية الاستخدام:**
```
اكتب @ ثم ابدأ بكتابة اسم المستخدم
استخدم الأسهم لاختيار الشخص
اضغط Enter أو Tab لتأكيد الاختيار
```

**مثال:**
```
@ahmad هل يمكنك مراجعة هذا؟
مرحباً @sara و @mohamed
```

---

### 4. الترجمة التلقائية
- ترجمة أي رسالة بنقرة واحدة
- كشف تلقائي للغة (عربي ↔ إنجليزي)
- عرض الترجمة مع الرسالة الأصلية
- إمكانية إخفاء الترجمة

**كيفية الاستخدام:**
- مرر الماوس على الرسالة
- اضغط على أيقونة اللغات
- سترى الترجمة تحت الرسالة الأصلية

---

### 5. إرفاق الصور
- رفع الصور مباشرة
- معاينة قبل الإرسال
- عرض بحجم كامل (lightbox)
- حذف الصورة قبل الإرسال

---

### 6. الحذف
- حذف الرسائل الخاصة بك فقط
- تأكيد قبل الحذف
- حذف فوري مع تحديث realtime

---

### 7. مؤشر الكتابة (Typing Indicator)
- عرض من يكتب حالياً
- يعمل بشكل فوري
- إخفاء تلقائي بعد التوقف عن الكتابة

---

### 8. المؤشرات المتصلة (Online Indicators)
- عرض الأعضاء المتصلين حالياً
- تحديث فوري عند الاتصال/الانقطاع
- عداد الأعضاء النشطين

---

### 9. طبقات الرسائل (Message Layers)
- **عادي (Standard)** - الرسائل العادية
- **مهم (Upper)** - للرسائل المهمة (محدود للمسؤولين)
- **ظل (Shadow)** - رسائل خاصة مرئية لأشخاص محددين

---

### 10. العقد (Nodes)
- ربط الرسائل بمواضيع محددة
- تنظيم أفضل للنقاشات
- فلترة الرسائل حسب العقدة

---

## Push Notifications

### الميزات:
- إشعارات فورية حتى عند إغلاق التطبيق
- دعم الإجراءات السريعة:
  - **فتح** - يفتح الرسالة/المجموعة
  - **رد سريع** - الرد مباشرة من الإشعار
  - **تجاهل** - إغلاق الإشعار
- Badge counter على أيقونة التطبيق
- اهتزاز مخصص للإشعارات المهمة

### أنواع الإشعارات:
1. **رسالة جديدة** - عند استلام رسالة
2. **إشارة (Mention)** - عند الإشارة إليك
3. **رد** - عند الرد على رسالتك
4. **تفاعل** - عند التفاعل مع رسالتك
5. **دعوة لمجموعة** - دعوة جديدة
6. **قرار جديد** - تصويت جديد
7. **ذاكرة جماعية** - ملخص جديد

## API المستخدمة

### إرسال رسالة مع إشارات:
```typescript
POST /api/messages/send-with-mentions
Body: {
  groupId: string
  content: string
  mentionedUsers: string[] // user IDs
}
```

### إرسال push notification:
```typescript
POST /api/push/send
Body: {
  userIds: string[] // multiple users
  title: string
  body: string
  url: string
  data: {
    type: "mention" | "message" | "reaction"
    groupId: string
    priority: "high" | "normal"
  }
}
```

## نصائح الاستخدام

### للحصول على أفضل تجربة:
1. **فعّل الإشعارات** من إعدادات المتصفح
2. **ثبّت التطبيق** على الشاشة الرئيسية (PWA)
3. **استخدم الإشارات** للفت انتباه أعضاء محددين
4. **استخدم Nodes** لتنظيم النقاشات المختلفة
5. **جرب الطبقات** لتمييز الرسائل المهمة

## الاختصارات

- **Shift + Enter** - إرسال الرسالة
- **@** - بدء إشارة
- **مرر الماوس** - عرض أزرار التفاعل
- **نقرة طويلة (موبايل)** - عرض الخيارات

---

## المتطلبات التقنية

### للمتصفح:
- Chrome 90+ أو Edge 90+ أو Safari 15+
- دعم Service Workers
- دعم Push API
- دعم Notifications API

### للـ APK:
- Android 8.0+ (API 26+)
- Google Play Services (للإشعارات)

## الأمان والخصوصية

- جميع الرسائل مشفرة في قاعدة البيانات
- Push notifications تستخدم VAPID للأمان
- الإشارات تُرسل إشعارات مباشرة فقط للمُشار إليهم
- لا يمكن حذف رسائل الآخرين
- الرسائل المحذوفة لا يمكن استرجاعها
